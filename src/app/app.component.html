1. Crie três novos componentes: GameControl, Odd e Even
  2. O componente GameControl deve ter botões para começar e parar o jogo
  3. Quando começar o jogo, um evento (guardando um número incrementado) deve ser emitido a cada segundo (ref = setInterval())
  4. O evento deve ser escutado de fora do componente
  5. Quando parar o jogo, nenhum evento deve ser mais emitido (clearInterval(ref))
  6. Um novo componente Odd deve ser criado para cada número ímpar emitido, o mesmo deve acontecer para o Even Component (nos números pares)
  7. Produza Ímpar - NÚMERO ou Par - NÚMERO nos dois componentes
